Reports

The Reports page allows you to set up reports for your projects.

The report is sent by email in the html/pdf format.

To add a report:

  • 1. Specify a report name.
  • 2. HTML or PDF report type.
  • 3. Select the domain for which the report will be generated.
  • 4. Select a report template.
  • 5. Select the report frequency: daily, weekly or monthly.
